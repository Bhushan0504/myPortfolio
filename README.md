## ğŸš€ Getting Started

Follow these steps to run the project locally on your machine.

### Prerequisites

-   [Node.js](https://nodejs.org/) (v18 or higher)
-   [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Bhushan0504/myPortfolio.git
    cd myPortfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open in your browser**
    Visit `http://localhost:5173` to view the app.

## ğŸ“¦ Deployment

This project is configured to deploy automatically to **GitHub Pages**.

### How it works
The `gh-pages` branch is used to host the production build of the website.

1.  **Build the project:**
    ```bash
    npm run build
    ```
    This compiles the code into the `dist` folder.

2.  **Deploy to GitHub Pages:**
    ```bash
    npm run deploy
    ```
    This command pushes the contents of the `dist` folder to the `gh-pages` branch.
